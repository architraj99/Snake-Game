# Snake Game

A Snake Game built by me

The player has to control snake using the arrow keys and try to collect as much food as possible.
Each food item increases the score and makes the snake longer. 
The game ends if the snake hits the wall or collide on itself...

## Features

* Snanke movement can be controlled through arrow keys
* In the game area food can be generated anywhere
* Score tracking (  1 point for 1 food )
* Snake will grow each time it eats the food
* If the snake collides on wall... the game will end
* Or if the snake collides on itself then there will be game over
* Game over screen
* Restart button
* High score 
* Increasing game speed as the score grows

## How to Play

1. Click the Start Game button.
2. Use the arrow keys to move the snake.
3. Collect food to increase your score.
4. Avoid hitting the walls.
5. Avoid running into your own.
6. Try to beat your high score.
